# FitnessMonster
fitNessMonster
Overview
fitNessMonster is a lightweight Java fitness tracking application built as an early prototype for a larger concept. The project focuses on application structure, data persistence, and user customization rather than UI or advanced features.
This application was designed to test ideas and architecture before expanding into a more fully featured fitness platform.

Features
Workout tracking with Cardio and Weights categories
Persistent workout history using file storage
Customizable user settings including date format and units
User profile management with saved personal data
All data persists between sessions using simple, human readable files.

Technical Highlights
Java console application
Object oriented design with clear separation of concerns
File based persistence using CSV and properties files
Easily extendable architecture suitable for future UI or mobile development

Purpose
This project serves as a proof of concept to validate design decisions and core functionality before investing in a larger scale application.

Status
Early prototype. Functionality and structure may evolve as the project expands.
